# 🌐 Keen Keeper App

A modern React-based web application to manage and track your personal relationships efficiently.
Stay connected with your friends through smart interaction tracking, analytics, and reminders.

---

## 🚀 Live Demo

🔗 [https://keen-keeper-shawon.netlify.app/](https://keen-keeper-shawon.netlify.app/)

---

## 📌 Project Overview

Keen Keeper helps you maintain meaningful connections by tracking interactions like calls, texts, and video chats. It provides a clean UI and insightful analytics to visualize your communication habits.

---

## 🛠️ Tech Stack

* ⚛️ React (Frontend)
* 🎨 Tailwind CSS (Styling)
* 📊 Recharts (Analytics Visualization)
* 🔔 React Toastify (Notifications)

---

## ✨ Key Features

* 📞 **Track Interactions**
  Add Call, Text, and Video activities for each friend

* 📊 **Analytics Dashboard**
  Visualize interaction data using Pie Chart

* 🧠 **Smart Duplicate Prevention**
  Prevents adding the same interaction twice

* 📅 **Timeline History**
  View all past activities with sorting & filtering

* 🚫 **Empty State Handling**
  Shows "No Data" UI when no interactions exist

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/your-username/friendship-tracker.git
```

Go to project directory:

```bash
cd friendship-tracker
```

Install dependencies:

```bash
npm install
```

Install required libraries:

```bash
npm install react-router-dom
npm install recharts
npm install react-toastify
```

Run the project:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
